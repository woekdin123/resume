Project: ARIMA and LSTM for Streamflow Time Series Forecasting

Overview: This project focuses on forecasting reservoir inflows using historical streamflow data. We employed statistical and machine learning models to improve the accuracy of inflow prediction essential for hydropower resource management.

Models Implemented:

  • ARIMA (Autoregressive Integrated Moving Average)
  • SARIMA (Seasonal ARIMA)
  • Bidirectional LSTM (Long Short-Term Memory)

Data Understanding:

We worked with monthly inflow data (in MCM). The time series was decomposed into:

  • Original: Raw streamflow data
  • Trend: Long-term pattern
  • Seasonality: Recurring annual patterns
  • Residuals: Irregular fluctuations

ARIMA Model:

A statistical model requiring stationary data. The final parameters were (p, d, q) = (1, 0, 1).
Performance: R-Squared = 0.29, RMSE = 195.19

SARIMA Model:

Enhanced version of ARIMA accounting for seasonality.
Performance: R-Squared = 0.4955, RMSE = 268.61

LSTM (Bidirectional):

A deep learning model that captures long-term dependencies effectively.
Performance: R-Squared = 0.8386

Conclusion:

While SARIMA improved over ARIMA due to seasonal adjustments, Bidirectional LSTM provided the highest accuracy in forecasting. Its ability to model complex time series makes it the most reliable choice for future reservoir inflow predictions.
